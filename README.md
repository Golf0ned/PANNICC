# PANNICC

Prototype And Non-Negligibly Instructive C Compiler

## Installation

Clone the repo:

```
git clone git@github.com:Golf0ned/PANNICC.git
cd PANNICC
```

To build and run tests, run:

```
make
```

## Usage

To view the help message:

```
./build/bin/pannicc
```
